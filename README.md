#Игра "Камень ножницы бумага"

## Описание

Консольная игра 


## Как играть
Для запуска в директории с игрой наберите python game.py
дальше следйте подсказкам(меню) в игре


## Функциональность

Игра работает через консоль, для выбора значений "камень, ножницы,бумага" используйте клавиатуру
для выхода нажимайте 0 если хотите посмотреть результаты после выхода из игры смотрите файл scores.json в директории с игрой


## Требования
Для работы игры требуются стандартные библиотеки python - random, json,file
